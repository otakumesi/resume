# 個人的に興味がある技術

## 情報検索技術
現在のありふれた検索サイトには「検索するために『検索をするテクニック』を必要とする」という課題があると考えている。  
得たい情報を手に入れるためには、検索の仕方を習得する必要がある。  
これによって、個人間に情報格差を生み出していると感じる機会があった。  
そのため、この問題を解決に導く研究をしたい。  
  
この問題には既存の情報検索システムに使われる技術の以下の部分に課題があると思っている。
  * ユーザーインターフェース
  * クエリ解釈
  * ランキングアルゴリズム
